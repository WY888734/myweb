步骤 7：点亮测试与收尾

一、点亮前最后检查

最终清单 (Final Check)：
  • 显示接口： 显示器线是否插在显卡接口上？（千万别插主板上！）
  • 电源开关： 电源背部的 I/O 开关 是否已拨至 "I" 位置？
  • 异物排查： 检查机箱内是否有散落的螺丝或扎带剪下来的碎屑，防止短路。
  • 内存安装： 再次按压内存，确保两端的扣具都已完全锁死。

三、关键状态排查 (EZ Debug LED)

开机后请观察主板边缘的小灯：

CPU 灯亮

通常是 CPU 供电没插好、针脚弯曲或需要更新 BIOS。

DRAM 灯亮

内存没插紧或不兼容。尝试拔掉一根内存进行单通道启动测试。

VGA 灯亮

显卡未正确识别。检查显卡金手指是否插好，以及供电线是否插紧。

BOOT 灯亮

恭喜！ 硬件通过自检。目前只是因为没装系统而报错。

四、成功点亮后的操作

  • 1. 进入 BIOS： 重启电脑，反复狂按 Del 或 F2 键。
  • 2. 开启 XMP (DOCP)： 在 BIOS 中开启 XMP，让内存运行在标注的 3200MHz+ 高频。
  • 3. 检查温度： 确认 CPU 待机温度在 30-50°C 左右。如果过高，说明散热器没压紧。
  • 4. 盖上侧板： 确认一切正常后，锁好机箱两侧盖板。
